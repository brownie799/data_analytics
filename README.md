 🚀 CryptoAI Analytics Dashboard

A modern AI-powered crypto trading analytics dashboard built using Python and Machine Learning.
This project analyzes historical trading data to uncover profitability trends, trader behavior, and market insights through interactive visualizations.
## ✨ Features
📊 Interactive trading analytics dashboard
💰 Profit & loss analysis
📈 Coin-wise performance tracking
⚡ Buy vs Sell behavior insights
🤖 Machine Learning-based trade prediction
🏆 Top trader performance analysis
🌙 Modern dark-themed visualizations
## 🛠️ Tech Stack
Python
Pandas
Plotly
Scikit-learn
Google Colab
 📌 Key Insights
Large trades create higher market volatility

Buy-side activity dominates profitable sessions
